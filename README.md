# Prakritee Dev — Animated Portfolio (GitHub Pages)

A responsive, ATS‑friendly, animated portfolio that acts like a resume but markets your skills as a web page.

## Quick start (GitHub website only)
1. Create a public repo named `yourusername.github.io`.
2. Upload all files in this folder to that repo.
3. Edit `index.html`: replace `your-username` with your actual GitHub username.
4. Replace `assets/Prakritee-Resume.pdf` with your real PDF resume (same filename).
5. Enable GitHub Pages: Settings → Pages → Deploy from a branch → `main` + `/ (root)` → Save.
6. Wait ~1–2 minutes; your site goes live at `https://yourusername.github.io/`.

## Customize
- Edit content in `index.html` (Experience, Projects, Skills, Certifications).
- Update JSON‑LD block with your links.
- Replace `assets/og-image.png` for better social previews.
- Colors/animation in `styles.css`.

## Print to PDF
Use your browser File → Print → Save as PDF for a clean one‑page resume export.
